# AutoPasteTool
 Tóm tắt tính năng
✅ Giao diện gọn gàng, cân đối tỷ lệ 60:40
✅ Tùy chọn tốc độ dán (stick button), mặc định 250ms
✅ START → Khóa vùng nhập liệu
✅ EDIT → Mở khóa vùng nhập liệu
✅ ESC → Bắt đầu/Dừng dán (kể cả khi không focus ứng dụng)
✅ RESET → Xóa dữ liệu nhập
✅ Đếm số dòng nhập liệu theo thời gian thực
